# 3. 画面遷移

## 1. ログイン

```mermaid
flowchart LR

Signup[サインアップ<br>P101105]
Login[ログイン<br>P101101]
Mypage[マイページ<br>P101103]
Logout[ログアウト<br>P101102]
Error[汎用エラー<br>P101104]

Signup -->|ユーザ登録完了| Login
Login -->|認証成功| Mypage
Login -->|認証失敗| Login
Login -->|サインアップへ| Signup

Mypage -->|ログアウト| Logout
Logout -->|ログイン画面へ| Login

Mypage -->|エラー| Error
Error -->|戻る| Mypage
```
